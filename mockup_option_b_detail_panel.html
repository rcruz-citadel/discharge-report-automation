Option B: Pros & Cons

Pros: Clean table — status is just a pill badge, no clutter. Detail panel gives room for notes, diagnosis info, and a confirmation button (no accidental saves). Easier to build reliably in Streamlit (expander or container below the table). Shows extra context (diagnosis, disposition, LOS) that helps the outreach call.

Cons: Extra click to open the panel. Slightly slower for bulk processing if you need to update 20 rows in a row. Only one row visible at a time in detail view.
